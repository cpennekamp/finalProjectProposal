Battleship

Strikes from the Dark

By Connor Pennekamp

Teams Members

Connor Pennekamp, Nicholas Rohmann, Joseph Koenig, Lama Abdelhamid

Application Overview

User will play Battleship against a computer which has three different sized battleships. The grid will be 5x5 and the user will have unlimited guesses. Depending on the hit to miss ratio, the user will recieve a score out of 25.

Technology Overview

This will be made using Python. It will be able to be played via a users terminal.
